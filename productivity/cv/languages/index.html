High Speed Trading System

Developed a low-latency trading platform with optimized performance. Implemented advanced algorithms for market analysis and automated trading decisions.

Components

  • Frontend - UI for configuring and deploying algorithms and data visualization
  • Backend - For supporting the frontend
  • Live - Node.js server that keeps on running, listens to the prices, runs them through the strategies and takes buy/sell/hold decisions
  • Mockmarket - WebSocket server that simulates live market prices for testing
  • deploy-ssh.js - A simple but useful script for uploading the changes to an AWS virtual machine and restart the server
  • start.sh/start.bat - Useful script for starting all the components in a single command
